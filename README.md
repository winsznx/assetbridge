# AssetBridge

Cross-chain asset migration protocol with validator consensus on Base and Stacks blockchains.

## Features

- Initiate cross-chain migrations
- Validator-based security
- Track migration status
- Multi-chain support
- Asset ID mapping

## Tech Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Base**: Solidity 0.8.20, Foundry, Reown wallet
- **Stacks**: Clarity v4, Clarinet, @stacks/connect

## Getting Started

```bash
pnpm install
pnpm dev
```
